# react18 + antd4.24 制作管理系统

## 初始化项目

```bash
# 创建新项目
npx create-react-app react-antd-demo
cd react-antd-demo
# 安装依赖
npm i
npm i react-router-dom axios nprogress antd@4 @ant-design/icons
```

### 依赖说明

| 插件              | 说明         |
| :---------------- | :----------- |
| nprogress         | 进度条       |
| axios             | 网络请求     |
| antd@4            | UI框架第4版 |
| @ant-design/icons | antd的图标库 |
| react-router-dom  | 路由         |


## 项目启动
```bash
npm start
```
